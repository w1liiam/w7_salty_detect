Salty Chat
Salty Chat is our in-game VOIP replacement and is needed to play on SunriseRP.

It give us a lot more possibilities, which currently allow such things as:
  • Better voice quality
  • Proximity chat unique to each player

Salty Chat has to be installed and running to play on our servers.

If you cannot join the appropriate server channel, please join and submit a support ticket, so that a support team member can help you.

You can find more information and download links at: Discord.gg/SRP

Press your Push to talk key to close the Salty Chat alert screen if you have moved into the appropriate VOIP channel in Teamspeak
